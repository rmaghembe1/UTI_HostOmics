# UTI HostOmics v1.0.0

## Release scope

This is the first frozen public reproducibility release accompanying the
mSystems-targeted manuscript:

**Host endocrine-metabolic-immune remodeling across recurrent and pregnancy-associated uropathogenic Escherichia coli infection**

## Included

- curated endocrine-metabolic-immune module definitions;
- public GEO accession manifest;
- phase-resolved analysis scripts with portable path substitution;
- selected metadata, reports, and source-value tables;
- eight assembled main figures;
- mSystems-targeted manuscript;
- supplemental workbook and legends;
- environment and reproducibility documentation.

## Data policy

Raw GEO data and large processed matrices are not redistributed. Source data
are reacquired from GSE112098, GSE186800, GSE252321, and GSE280297.

## Reproducibility constraints

Each dataset was analyzed in its native species, tissue, and experimental
design. Raw expression values were not pooled across studies. Cross-context
integration used standardized module effects, recurrence, directional
concordance, and tissue coherence.

# Data access

The repository does not redistribute raw GEO data.

The four source datasets are publicly available from the NCBI Gene Expression
Omnibus:

- GSE112098
- GSE186800
- GSE252321
- GSE280297

The `accessions.tsv` file records their analytical roles. Repository scripts and
manifests are intended to support reacquisition and reconstruction from the
public sources. Derived non-restricted tables needed to inspect the reported
results are provided under `results/` and `supplemental/`.

# Contributing

This repository accompanies a manuscript and preserves a controlled analysis
release. Contributions that improve portability, documentation, or error
reporting are welcome.

Before opening a pull request:

1. Do not add raw GEO data or restricted data.
2. Do not commit local machine paths, credentials, tokens, or private keys.
3. Keep generated files below GitHub's file-size limits.
4. Preserve the native species and tissue design of each dataset.
5. Describe any analytical change and its effect on reported results.
6. Run Python and shell syntax checks for modified scripts.

Scientific changes affecting manuscript results should be discussed in an issue
before implementation.
